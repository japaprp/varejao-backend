Política de Privacidade

Como coletamos, usamos e protegemos os dados dos usuários.

Início Exclusão de dados

1. Dados coletados

Podemos coletar nome, e-mail, CPF, telefone, endereço e dados de pedidos para viabilizar autenticação, entrega e suporte.

2. Uso das informações

Os dados são utilizados para cadastro, login, histórico de compras, fidelidade, atendimento e melhoria da experiência.

3. Compartilhamento

Não vendemos dados pessoais. Podemos compartilhar dados apenas com provedores necessários para operação do serviço (ex: pagamento).

4. Segurança

Aplicamos controles de autenticação e proteção de acesso. Ainda assim, nenhum sistema é 100% imune a falhas.

5. Direitos do titular

Você pode solicitar correção ou exclusão dos seus dados pela página de exclusão:
